# Introducción a `std::vector`

`std::vector` es el **arreglo dinámico por defecto en C++**.

- **array (arreglo)** → puede almacenar varios valores/objetos del mismo tipo  
- **dynamic (dinámico)** → su tamaño puede cambiar en tiempo de ejecución  
---

# Inicialización / Acceso

```cpp
#include <vector>
#include <iostream>

int main () {
using std::cout;
std::vector<int> v {2, 7, 9};
cout << v.size() << '\n';
cout << v[0] << '\n';
cout << v[1] << '\n';
v[1] = 4;
cout << v[1] << '\n';
cout << v.front() << '\n';
cout << v.back()  << '\n';
}

````

### Explicación línea por línea

* `#include <vector>` → archivo de cabecera
* `std::vector<int> v {2, 7, 9};` → inicializa con 3 elementos
* `v.size()` → número de elementos (3)
* `v[0]` → primer elemento
* `v[1]` → segundo elemento
* `v[1] = 4;` → asignar nuevo valor
* `v.front()` → primer elemento
* `v.back()` → último elemento

### Salida esperada

```
3
2
7
4
2
9
```

⚠ Cuidado con errores comunes en la inicialización de `std::vector`.

# Agregar elementos

## `vector<T>::push_back(elemento)`

Agrega un elemento de tipo T al final del vector.

```cpp
std::vector<int> v;
// [] 
// 0
cout << v.size() << '\n';
// [2]
//  1
v.push_back(2);
cout << v.size() << '\n';
// [2][7]
//  1  2
v.push_back(7);
cout << v.size() << '\n';
// [2][7][9]
//  1  2  3
v.push_back(9);
cout << v.size() << '\n';
```

---

# Redimensionar

## `vector<T>::resize(nuevo_tamano, valor_relleno=T{})`

```cpp
std::vector<int> v {1,2};

v.push_back(3);
cout << v.size() << '\n';

v.resize(6, 0);
cout << v.size() << '\n';
```

### Explicación

* Inicial: `{1,2}`
* Después de `push_back(3)` → `{1,2,3}`
* Después de `resize(6, 0)` → `{1,2,3,0,0,0}`

### Salida

```
3
6
```

# Eliminar elementos (al final)

## `pop_back()`

Elimina el último elemento.

## `clear()`

Elimina todos los elementos.

```cpp
std::vector<int> v {1,2,3,4,5,6};

cout << v.size() << '\n';

v.pop_back();
cout << v.size() << '\n';

v.pop_back();
cout << v.size() << '\n';

v.clear();
cout << v.size() << '\n';
```

### Evolución del vector

```
{1,2,3,4,5,6} → tamaño 6
{1,2,3,4,5}   → tamaño 5
{1,2,3,4}     → tamaño 4
{}            → tamaño 0
```

# Las copias son profundas (Deep Copy)

`std::vector` es un tipo regular.

Se comporta como un `int` en los siguientes aspectos:

* Copia profunda → se crea un nuevo vector y se copian todos los elementos
* Asignación profunda → se copian todos los elementos
* Comparación profunda → compara los valores internos
* Propiedad profunda → al destruir el vector, se destruyen sus elementos

La mayoría de los tipos estándar de C++ se comportan así.

---

## Ejemplo

```cpp
std::vector<int> a {1,2,3,4};
std::vector<int> b = a;  // copia a → b

if (a == b) cout << "equal\n";

a[0] = 9;

cout << b[0] << '\n';

if (a != b) cout << "different\n";
```

### Resultado

```
equal
1
different
```

Explicación:

* `b` es una copia independiente
* Modificar `a` no afecta a `b`

⚠ Copiar vectores grandes puede ser costoso en tiempo si contienen muchos elementos.

# Ejercicio práctico

## Ejercicio 1 – Manipulación básica

Escribe un programa que:

1. Cree un `std::vector<int>` vacío.
2. Agregue los números 10, 20 y 30 usando `push_back`.
3. Imprima:

   * el tamaño del vector
   * el primer elemento
   * el último elemento
4. Cambie el segundo elemento a 99.
5. Elimine el último elemento.
6. Imprima nuevamente el tamaño final.

1. Crea un vector `a` con los valores `{5, 6, 7}`.
2. Crea otro vector `b` como copia de `a`.
3. Cambia el primer elemento de `a` a 100.
4. Imprime ambos vectores.
5. Explica qué ocurrió.
